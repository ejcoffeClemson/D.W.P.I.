Dealing with PPM Images

Description: Implemented a program that removes noise by averaging 10 image files pixels and the ability to remove a person from an image by taking the median of 9 image files pixels

How to Run:

	Meadian- In the terminal, type "make" to compile the program and then type "make run" to run the program

	Average- Firstly, open the makefile and look under the run: build section, change the word "medain" to "average", finale
	perform the same steps as in Meadian section above
